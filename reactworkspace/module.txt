
컴포넌트 전환을 위한 Router 사용을 위해 설치
npm install react react-dom react-router-dom

리액트 기반의 사용자 인터페이스 라이브러리
npm install @mui/material @emotion/react @emotion/styled @mui/icons-material react-icons

애플리케이션 전역에 영향을 주는 상태를 관리하기 위해 설치
npm install zustand

폼 상태 관리 및 유효성 검사
npm install react-hook-form @hookform/resolvers yup

서버에 HTTP 비동기 요청을 보내기 위해/ 브라우저 쿠키 관리 (토큰 저장/삭제 등)  /사용자에게 실시간 알림 메시지를 띄워주는 라이브러리
